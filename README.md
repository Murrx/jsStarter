# jsStarter
A starter kit for Javascript projects

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running
To run the webpack-dev-server
```sh
node run start:dev
```

```sh
node dist/bundle.js
```
